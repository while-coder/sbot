# Wiki 知识库

侧栏 → **知识库（文档）** → 新建

Wiki 是结构化的、人工编纂的知识页面库（标题 + 内容 + 标签）。Agent 可通过内置工具搜索、读取、写入、更新页面 —— 非常适合项目文档、值班手册、FAQ 和团队共享参考资料。

## 配置项

| 字段 | 说明 |
|-------|-------------|
| 名称 | Wiki 标识 |
| 向量模型 | 可选 —— 设置后启用关键词 + 语义混合检索；不设置则退回到纯关键词检索 |

## 工作原理

页面通过两种方式建立索引：

1. **关键词**（始终可用）—— 标题 + 内容 + 标签全文匹配
2. **语义**（配置向量模型时）—— 向量相似度，用于模糊 / 概念查询

查询时两种信号会合并 —— 因此 Agent 搜索 "deploy steps" 时，既能命中标题完全匹配的页面，也能命中语义相关的 "release procedure" 页面。

## Agent 工具

Wiki 被分配到会话 / 渠道后，Agent 自动获得以下工具：

- `wiki_search` —— 按关键词与 / 或语义相似度查询
- `wiki_read` —— 按 id 读取整页
- `wiki_create` —— 新建页面（标题、内容、标签）
- `wiki_update` —— 修订已有页面
- `wiki_delete` —— 删除页面

## 分配

Wiki 可在多个层级分配（最具体的优先）：

- **Agent 级默认值** —— 使用此 Agent 的所有会话 / 渠道继承
- **会话级** —— 在聊天会话中覆盖
- **渠道级** —— 在渠道中覆盖
