# Models

Sidebar → **Models** → New

Fill in provider, API key, base URL, and model name.

## Supported Providers

| Provider | Notes |
|----------|-------|
| OpenAI | GPT-4o, GPT-4, etc. |
| Anthropic | Claude series |
| Google Gemini | Gemini Pro, Flash |
| Ollama | Local models |
| OpenAI-compatible | Azure OpenAI, Groq, Mistral, DeepSeek, etc. |

Any endpoint that implements the OpenAI chat completions API can be used as a provider.
