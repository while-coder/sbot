# Skills

Sidebar → **Skills**

Skills are installable prompt modules (Markdown files) that extend agent capabilities with specialized knowledge and workflows.

## Storage

Skill files are stored in `~/.sbot/skills/`.

## Installation

- Search and install from remote hubs (Clawhub, skills.sh) on the Skills page
- Or drop Markdown files manually into the skills directory

## Assignment

In an agent → Skills tab:
- Select specific skills to load
- Or leave empty to load all available skills
