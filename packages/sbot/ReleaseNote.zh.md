## v0.0.7

### 新功能

- **压缩文件工具**：新增内置压缩文件操作工具，支持压缩、解压、列出内容及读取压缩包内文件
- **二进制文件读取工具**：新增在 Agent 工作流中读取二进制文件的工具
- **Lark 发送文件/图片**：Lark 渠道现支持向用户发送图片和文件
- **Lark 接收文件/图片**：Lark 渠道现支持接收用户发送的图片、文件及其他媒体内容
- **find-skills Skill**：新增内置 Skill，用于发现 skills 目录中可用的技能
- **多份 Memory 支持**：Agent 现在可以同时为每个用户/会话加载并使用多份 memory 文件
- **Memory 管理界面**：Web 管理界面新增对单条 memory 内容的查看与管理功能
- **Docker 支持**：新增容器化部署支持

### 改进

- **Saver 保存格式**：优化 `AgentFileSaver` 的文件保存格式，输出更加清晰
- **Prompt 优化**：优化了各渠道在询问和发送文件场景下的提示词
- **UserService 重构**：统一并整理了各渠道的用户服务代码
- **AgentRunner**：清理内部 Agent 执行流程，改进提示词处理逻辑
- **Skills 目录**：新增对 skills 目录级别描述的支持，便于技能发现
- **Memory 存储结构**：将 memory 存储重构为数组格式，以支持多份 memory

### Bug 修复

- 修复渠道注册问题
- 修复网页端客户端显示问题
