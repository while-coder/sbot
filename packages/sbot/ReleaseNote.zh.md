# 发布说明

本次更新扩展了 sbot 的接入面：新增 3 个聊天渠道，内置内网穿透服务（无需额外反向代理即可对外暴露本地服务器），新增桌面 Helper 应用（用于需要内嵌浏览器的渠道登录），并显著提升了 MCP 集成的稳定性。

### 新增渠道

- **小爱音箱（小米）** —— 通过小米/小爱音箱与 Bot 对话，轮询间隔可配置。
- **钉钉** —— 一等公民的钉钉渠道，适用于企业群聊场景。
- **QQ（OneBot v11）** —— 接入任意兼容 OneBot v11 的 QQ Bot 框架。

### 内置内网穿透

新增内网穿透服务，可在管理后台一键将本地服务器暴露到公网，提供三种方式：Cloudflare Quick（零配置）、Cloudflare Token（自有命名隧道）、Localtunnel。所需的 `cloudflared` 二进制会在首次使用时自动下载。

### 桌面 Helper 应用

新增基于 Tauri 的桌面 Helper，承载需要内嵌浏览器的渠道登录流程。首个落地的是小爱 cookie 登录 —— 在 Helper 窗口完成一次登录，sbot 自动接手会话。

### MCP 连接更稳定

远端 MCP 服务（HTTP / SSE / streamable HTTP）在重启或网络抖动时常会丢失 session，导致工具调用失败。本次更新可识别这类失效连接并自动重连，对 Agent 透明，不再因临时故障打断对话。

### 工具输出处理

超长的工具输出会在送入模型前自动截断，并支持按 Agent 配置上限。`read` 文件工具同步重写，使大文件在长对话中表现更可控。

### 飞书增强

新增"获取群组列表"能力，Agent 可直接管理飞书群会话。

### 调度器

定时任务现在支持禁用而不必删除，更方便地暂停和恢复周期性任务。

### 统一的渠道工具

所有渠道现在向 Agent 暴露同一套工具 —— 列出渠道/会话/用户、发送消息、发送文件 —— 多渠道 Agent 在不同平台间表现一致。

### 文档

文档站完成整体翻新，刷新主题并补齐全量中文翻译，覆盖每个 Guide 页面（Agents、Channels、Tools、MCP、Memory、Skills、Wiki、Heartbeat、Insight、Note、Savers 等）。
