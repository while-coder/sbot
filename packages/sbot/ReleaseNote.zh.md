本次主要更新如下：

### 新增

1. 新增意图过滤模式：渠道和会话配置可以选择“自动”“不过滤”“全部过滤”，会话配置仍可继承渠道默认值。
2. 结构化输出调用新增方法选项，可按模型能力选择 Function Calling、JSON Mode 或 JSON Schema，并支持 strict 配置，提升 OpenAI 兼容模型的适配能力。

### 优化

1. 后台渠道和会话配置页支持查看、编辑意图过滤模式；当选择不过滤或全部过滤时，会自动隐藏并清理不再需要的模型、提示词和阈值配置。
2. 意图识别改用 JSON Mode，并补充只返回 JSON 的约束提示；日志也会记录模型 ID 与真实会话名称，排查误过滤和模型兼容问题更方便。
3. App 发布流程中的 Release 标题改为 `app v版本号`，避免和 sbot 包发布混淆。

### 修复

1. 修复会话配置中意图过滤相关字段保存和更新不完整的问题。
2. 修复意图过滤使用线程 ID 作为会话上下文的问题，日志和会话识别现在会按实际会话解析。
