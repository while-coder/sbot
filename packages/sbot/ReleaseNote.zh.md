本次主要更新如下：

### 新增

1. 新增「议程（Agenda）」系统，统一替代原有的定时任务（Scheduler）和待办（Todo）。支持创建、查看、更新、完成、取消和手动触发，聊天和管理页面均可编辑。
2. 新增智能心跳（Smart Heartbeat），由 AI 判断当前是否适合主动发起消息，避免固定频率打扰。
3. 笔记新增混合搜索（HybridSearcher），同时利用关键词与语义检索，结果更精准。
4. Web 管理界面新增设置（Settings）接口与记忆（Memories）管理接口。

### 优化

1. 重构记忆（Memory）系统，采用新的存储与服务架构，写入更可靠，失败时自动重试，读写提示词更清晰。
2. 依赖注入容器改为同步初始化，启动更快、行为更可预测。
3. 优化心跳模块，新增节流与统一上下文构建，减少重复触发。
4. 优化 Web fetch 工具，正文抽取与错误处理更稳定。
5. 优化时间处理：统一时间工具类，议程移除显式时区字段，使用更直观。
6. 优化渠道数据同步流程与错误反馈。
7. 精简议程相关提示词与工具定义，模型行为更清晰，Token 占用更低。

### 修复

1. 修复多处议程同步问题，包括编辑后状态错误以及触发遗漏。
2. 修复工具调用中「当前时间 / 今天日期」前后不一致的问题。
3. 修复图片缩放在大尺寸或非标准输入下的异常。
4. 修复部分配置下渠道提示词加载异常的问题。
