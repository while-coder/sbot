Reads a media file (image, audio, PDF, etc.) and returns its content in multimodal format. For images (.png, .jpg, .gif, .webp, .bmp, .svg), returns an image content block. For audio (.mp3, .wav, .ogg, .flac, .aac, .m4a), returns an audio content block. For other binary files, returns base64-encoded text with metadata. Maximum file size 100KB. Path must be absolute. Use read for text files (source code, config, markdown, etc.).
