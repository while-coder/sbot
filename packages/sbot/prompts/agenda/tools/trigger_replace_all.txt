Replace the FULL active trigger list of an agenda item in one shot.

USE WHEN
- Schedule cleanup, deduplication, or normalizing several active triggers.
- The user describes the final complete reminder list ("以后就只在 9:00 和 18:00 提醒，去掉中午那个").

Pass exactly the triggers that should remain active. Each spec carries its own `action` (default notify) and `message` (default = item.content). Existing active triggers not in the list are disabled.

EDGE CASES
- triggers=[] → clear all active triggers (the item itself stays; existing dueAt is unchanged unless explicitly modified via agenda_update).
- For a single targeted edit, prefer agenda_trigger_update / agenda_trigger_add — replace_all is heavier and harder for the user to reason about.
