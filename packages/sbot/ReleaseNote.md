This release includes the following main changes:

### Added

1. Added file upload, file/folder deletion, folder creation, and direct file editing in the file explorer.
2. Added session search in the chat sidebar.

### Improved

1. Improved upload conflict handling with clear overwrite, skip, or cancel choices.
2. Improved the chat UI by remembering the right-side workbench panel state on desktop layouts.
3. Improved the Channels page so channels, sessions, and users are easier to view consistently.
4. Improved app release packaging.
5. Improved bundled dependency handling during release packaging.

### Fixed

1. Fixed an issue where the built-in terminal could be unavailable on some machines.
2. Fixed terminal failure feedback so sbot now reports a clear error instead of silently failing.
